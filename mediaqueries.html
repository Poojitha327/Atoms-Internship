Rain Detector Using Arduino

This project is designed to detect rainfall using a rain sensor module connected to an Arduino. It helps users in agriculture or households to act quickly when rain begins and harvest water effectively.

  • Uses a rain sensor to sense rain drops on its surface.
  • Alerts the user using an LED or buzzer when rain is detected.
  • Farmers can cover crops to avoid damage and collect rainwater for irrigation.
  • Encourages rainwater harvesting for household reuse.
  • Tools Used: Arduino Uno, Rain Sensor Module, Buzzer/LED, Jumper Wires, Breadboard

Web Scraping: Comparison of Cricket Teams using ESPN Cric-Info

This project automates data extraction from ESPN Cric-Info to compare cricket teams and player performance. It involves scraping data using Python and analyzing it with libraries like Pandas and Matplotlib.

  • Collected team stats, player scores, match histories, and rankings.
  • Data analyzed to compare team performance and generate insights.
  • Visualized performance trends using charts and graphs.
  • Tools Used: Python, BeautifulSoup, Requests, Pandas, Matplotlib
